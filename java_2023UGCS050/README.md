## To Run CLI APP 
```bash
docker compose up --build -d
```
- Then attach to the cli container to interact.

```bash
docker attach cli
```
## To Stop CLI APP 
```bash
docker compose down
```


## About

This is a simple command-line banking application written in Java.

It lets users create accounts, deposit and withdraw money, transfer funds between accounts, and check their balance. All interaction happens through the terminal.
